## Hastane Sıra Takip Sistemi

Bu proje, hastanelerdeki hasta sırasını gerçek zamanlı olarak takip etmek için geliştirilmiş bir web uygulamasıdır.
                                                                                                                                                                                                                                                                                                                
## Proje Amacı

Hastanelerde hastaların sıralarının ekranda güncel olarak gösterilmesini ve sıradaki hastanın çağrılmasıyla birlikte ekranın otomatik olarak güncellenmesini sağlamaktır.

## Kullanılan Teknolojiler

Backend
- Node.js
- Express.js
- SQL Server
- SQLite (log sistemi için)
- Socket.io (gerçek zamanlı veri aktarımı)

Frontend
- React
- HTML / CSS

Diğer
- REST API
- Git / GitHub
